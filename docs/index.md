# gig 文档

## 命令参考

### 项目操作

| 命令 | 说明 | 文档 |
|------|------|------|
| `gig add` | 添加模板规则 | [add](commands/add.md) |
| `gig ignore` | 忽略文件/文件夹（已跟踪的自动取消跟踪） | [ignore](commands/ignore.md) |
| `gig check` | 检查文件是否被忽略 | [check](commands/check.md) |
| `gig view` | 查看 .gitignore 文件 | [view](commands/view.md) |
| `gig doctor` | 诊断 .gitignore 健康状态 | [doctor](commands/doctor.md) |
| `gig refactor` | AI 整理优化 .gitignore | [refactor](commands/refactor.md) |
| `gig track` | 强制跟踪被忽略的文件 | [track](commands/track.md) |

### 模板管理

| 命令 | 说明 | 文档 |
|------|------|------|
| `gig template local` | 管理本地模板 | [template](commands/template.md) |
| `gig template github` | 管理 GitHub 官方模板 | [template](commands/template.md) |

### 配置与工具

| 命令 | 说明 | 文档 |
|------|------|------|
| `gig config` | 管理配置 | [config](commands/config.md) |
| `gig version` | 显示版本信息 | [version](commands/version.md) |
| `gig completion` | 生成 Shell 补全 | [completion](commands/completion.md) |
